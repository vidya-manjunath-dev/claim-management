# Insurance Policy & Claim Management System - Frontend

A React-based frontend application for managing insurance policies and claims.

## Tech Stack
- React 18
- Vite
- React Router v6
- Axios
- React Toastify

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Environment Variables

Create a `.env` file based on `.env.example`:

```
VITE_API_BASE_URL=http://localhost:8080/api
```
